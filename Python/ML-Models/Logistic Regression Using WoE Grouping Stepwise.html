# basic ks function
def ks_function(x):
    # taking a subset to keep things simple for functionalisation
    # bad flag can be a generic indicator whether the row belongs to distribution 1 or 2
    # score is an identifier of the distribution x - axis 
    # a weight has to be added if input data is already grouped up
    ##sample = df_post.loc[:,['bad_flag','score']].copy()
    ##x = df_post.loc[:,['score','bad_flag']]
    col1 = x.columns[0]
    col2 = x.columns[1]
    # sort to have right order
    #sample.sort_values(by='score')
    x.sort_values(by=col1)

    # grouping by score and dropping level to get columns
    # assuming no missing values count gives # of distribution1+2, sum just # of distribution 1
    # count-sum gives # of distribution 2
    distr = x.groupby(col1).agg(['count','sum'])
    distr.columns = distr.columns.droplevel(0)
    distr['dis1'] = distr['sum']
    distr['dis2'] = distr['count'] - distr['sum']

    # calculating cumulative percentages
    distr['dis1_cum_sum'] = distr.dis1.cumsum()
    distr['dis1_perc'] = distr.dis1 / distr.dis1.sum()
    distr['dis1_cum_perc'] = distr.dis1_cum_sum/distr.dis1.sum()

    distr['dis2_cum_sum'] = distr.dis2.cumsum()
    distr['dis2_perc'] = distr.dis2 / distr.dis2.sum()
    distr['dis2_cum_perc'] = distr.dis2_cum_sum/distr.dis2.sum()

    # calculating KS
    distr['KS'] = distr['dis1_cum_perc'] - distr['dis2_cum_perc']
    KS = distr.KS.max()
    return KS

Stepwise Regression¶

Output is a Dataframe containing list of variables, coefficients, training Gini and training KS for each iteration. The number of iterations is set manually.

Building a first model with the 2 variables with highest univariate Gini or an alternative set of variables chosen as input. Subsequent iterations use variables from the variable pool in the order they appear in it.

Variables are excluded from the model if they have an absolute coefficient below threshold (0.05) or correlate to another variable selected for regression. The correlated variable with the lower coefficients will be discarded.

The stepwise regression will include all variables possible. It will only include one additional variable per iteration, so the number of variables in the model may fall if many are found to be correlated or no longer contributing to the model.
